# Contributing

感谢你对 SnowmanNunu/Weather 项目的关注！

## 开发流程

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feat/your-feature`
3. 提交代码：`git commit -m 'feat: add something'`
4. 推送分支：`git push origin feat/your-feature`
5. 发起 Pull Request

## 代码规范

- PHP 8.0+
- PSR-12 代码风格
- PHPStan Level 8 零错误
- PHPUnit 测试覆盖新功能

## 提交信息规范

- `feat:` 新功能
- `fix:` 修复
- `refactor:` 重构
- `test:` 测试
- `docs:` 文档
